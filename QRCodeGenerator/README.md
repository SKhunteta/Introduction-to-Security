Instructions:

Requires installation of Python 3, and the libraries qrcode and Pillow.

Use pip3 to install them.
	
pip3 install qrcode
pip3 install Pillow

How to run:

To generate QR Code:

	python3 SecurityAssignment3.py --generate-qr

OR

To get One-Time Password:

	python3 SecurityAssignment3.py --get-otp
